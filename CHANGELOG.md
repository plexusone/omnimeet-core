## [v0.1.0] - 2026-07-04

### Highlights

- Initial release with unified abstractions for real-time collaboration platforms
- Provider-agnostic design supporting LiveKit with Daily and Zoom planned
- Agent skill interfaces for OmniAgent integration
- Voice integration types for STT/TTS via OmniVoice
